## ğŸ”§ HOW TO CUSTOMIZE

### Add New VR Projects
Edit `/bayt/script.js`, find `const projects = [...]`:
```javascript
projects.push({
    name: "Your Project Name",
    thumb: "images/your-thumb.jpg",
    url: "https://your-shapespark-url/"
});
```
Then: `git add . && git commit -m "Add new project" && git push`

### Change Login Credentials
Edit `/bayt/script.js`:
```javascript
const VALID_USERNAME = "NewUsername";
const VALID_PASSWORD = "NewPassword";
```

### Change Colors
Edit `/bayt/style.css`, `:root` section:
```css
--accent-blue: #YourColor;
--accent-cyan: #YourColor;
```

### Change App Name
Edit `/bayt/manifest.json`:
```json
"name": "Your App Name",
"short_name": "ShortName"
```
